# rust-web-view-demo

#### Uses
- [web-view](https://github.com/Boscop/web-view)
- [React](https://github.com/facebook/react/)
- [TypeScript](https://github.com/microsoft/typescript)
- [Parcel](https://github.com/parcel-bundler/parcel)

#### Debug
```shell
# js-src
yarn && yarn run build

# root
cargo run
```
